# Assets Required for App Store Build

Place the following files here before running `eas build`:

- icon.png          — 1024×1024 app icon (no rounded corners, iOS adds them)
- adaptive-icon.png — 1024×1024 Android adaptive icon foreground
- splash.png        — 1284×2778 splash screen (iPhone 14 Pro Max size)
- notification-icon.png — 96×96 white notification icon on transparent background

Brand color: #00A651 (Kaero green)
